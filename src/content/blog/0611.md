# 背景:

第一次参加线下的分享会时，貌似大家都在使用更加便捷的交流方式，只有我在使用古法的PPT交流看起来有点落伍了.可是自己是c++程序员，不太懂前端网页的一些内容，不知道如何做这件事情..

![打卡纪念](/images/blogimages/0611.png)

## 摸索的过程:

从零搭建，感觉难度太大，使用简单的界面感觉没有格调，于是花了一两天的时间，去github上去找了很多开源的个人主页；也不是漫无目的的寻找，当然了，当时也没有想的那么久远 **有数据库**、**界面炫酷**，第一点保证了后续博客扩展的可能、数据存储，第二点后续的主页排版以及布局调整. 还是非常的具有挑战性的.


### 实操:

- 1. 使用 git 克隆 homepage 的代码 (同理github搜索个人主页、或者 hoempage)
- 2. 使用 AI 慢慢修改为，你想要的样子，这里需要你有耐心，慢慢的修改；
- 3. 使用 vercel 部署 (这里是我的部署方式，你也可以使用国内的部署——阿里云、腾讯云 || cloudflare || github.io) 我之所以使用了这个部署，是由于，我尝试了很长时间的在 cloudflare & github.io 上部署均以失败告终，也可能是我的方法问题... 我们只叙述成功的方式.
